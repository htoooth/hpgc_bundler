# hpgc_bundler

This bundler include gdal, geos, proj4, postgis, postgresql, protobuf, openmpi and ssdb.

## How to use

`git clone https://github.com/htoooth/hpgc_bundler.git`

then `cd hpgc_bundler`

type `rake`
